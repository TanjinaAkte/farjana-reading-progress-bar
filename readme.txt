== Frequently Asked Questions ==

= Does it work with all WordPress themes? =

Yes! The plugin is designed to work with any WordPress theme that follows WordPress coding standards.

= Can I customize the colors? =

Absolutely! You can customize:
* Progress bar color
* Reading time box background
* Text colors
* Border colors
* And more!

= Will it slow down my website? =

No! The plugin is extremely lightweight and optimized. It adds minimal CSS and JavaScript, with zero impact on performance.

= Does it work on mobile devices? =

Yes! The plugin is fully responsive and works perfectly on all screen sizes - mobile, tablet, and desktop.

= Can I disable it on certain post types? =

Yes! You can choose which post types to display the reading progress on from the settings page.

= How is reading time calculated? =

Reading time is calculated based on Words Per Minute (WPM). The default is 200 WPM, but you can adjust it in settings.

= What is the "Shukriya" celebration popup? =

It's a beautiful reward popup that appears when readers finish your article, thanking them with personalized messages and confetti animation!

= How does mood detection work? =

The plugin analyzes your content for emotional keywords and displays appropriate messages (happy for positive content, empathetic for sad content).

= Can I translate the plugin? =

Yes! The plugin is translation-ready and currently supports English and Bengali, with more languages coming soon.

= Does it track user data? =

No! The plugin is 100% privacy-friendly and doesn't collect, store, or transmit any personal data.

= Can I hide the reading time but keep the progress bar? =

Yes! You can independently enable/disable the reading time display and progress bar.
= Is there a pro version? =
No! All features are 100% FREE forever. We want to support the WordPress community with all premium features at no cost!

= How do I get support? =

Visit the plugin's support forum on WordPress.org or contact us directly through our website.
